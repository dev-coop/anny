Connection

Neuron. Connection

new Connection(source, target, weight)

The connection between two Neurons. Connections are unidirectional.
Parameters:
Name Type Description
source object Neuron that will send its output to the `target` Neuron.
target object Neuron that will get its input from the `source` Neuron.
weight number The strength of the connection. Meaning, how much of the `source` output is passed to the `target` Neuron input.

weight :number

The weight is used as a multiplier for two purposes. First, for activation, when transferring the output of the `source` Neuron to the input of the `target` Neuron. Second, during training, calculating the total error delta.
